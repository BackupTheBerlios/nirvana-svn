Nirvana

What is Nirvana?

Nirvana is the new approach to produce brand new alternative way to Web Browsing under BeOS/Haiku. Our main goal is bring free, fast browser which is coded in BeOS-native C++ style and utilizes proven codebase, with support for modern technologies like CSS, JavaScript, XHTML and ability to render most of current internet pages.

What is code based on?

We intend to follow Apple engineers who created Safari web browser, based on KHTML rendering engine (and KJS for JavaScript support).

The Scope of work

In order to use WebCore (KHTML adopted by Apple and abstracted from Qt drawing libraries) engine under BeOS we need to implement KBE abstraction layer (like KWQ [quack] layer done in Apple) over Interface Kit classes (BView etc). KWQ uses obects derived from NSObject for commucation with host OS components. In BeOS we need to start implementation of similar layer named KBE [ka:be].

Why not Gecko or Why to join us?

The main point not to use Gecko engine is its size, complexity and problems with maintenance. Also, there is already Mozilla (BeOS) team on the road, and they are porting Mozilla code from main branch regularly. If you prefer Gecko you can use Mozilla/BeOS which already exists or you wish to create new alternative for Web-browsing in BeOS you can join Nirvana to help us with porting KHTML.

Source Code

If you want help try to checkout source code read README.BeOS and follow TODO list and ask questions on google group. You can observe source code in public SVN repository at:

svn co svn://svn.synrc.com/trunk/nirvana

Public web access to svn repository can be obtained from svn://svn.berlios.de/nirvana. Anonymous check out source code as:

# svn co svn://svn.berlios.de/nirvana
